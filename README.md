3. 다운로드가 완료되면, ZIP 파일을 다음 경로에 풀어주세요 D:\WorkSpace\Robustness
4. 최종적으로 D:\WorkSpace\Robustness 파일 내에 아래 파일처럼 구성돼 있으면 됩니다.
